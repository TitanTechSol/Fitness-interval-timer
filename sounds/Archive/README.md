# Message Archive

This folder contains the original default message files for the Fitness Interval Timer.

## Purpose

These files serve as backups of the original messages that came with the application. If you edit or customize your messages and want to revert to the defaults, you can use the "ğŸ”„ Restore Original Messages" button in the application settings.

- **Audio3.md** - Original punishment messages
- **Audio4.md** - Original encouragement messages

## Manual Restoration

You can also manually copy these files back to the parent `sounds/` folder if needed:

```bash
cp Archive/Audio*.md .
```

**Warning**: This will overwrite your current message files!
